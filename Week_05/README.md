### AVL  
又叫平衡二叉树，同时是一种二叉搜索树，是为了防止普通的二叉搜索树退化为链表而进行设计的  
对抗措施主要是维护一套平衡因子，平衡因子主要利用左右子树的高度差进行计算，得到因子之后   
再利用四种旋转操作进行平衡优化  
不足: 需要额外存储信息，如果插入数据频繁则调整较为频繁  


### 红黑树  
红黑树则可看作是为了弥补AVL中那亮点不足而存在的，由平衡因子维护高度差妥协到左右子树的  
高度差小于两倍  
特点:  
1.每个节点或红或黑   
2.根节点为黑    
3.叶子节点为黑    
4.没有相邻的红色节点  
5.任意节点到每个叶子节点的所有路径中包含相同数量的黑色节点
